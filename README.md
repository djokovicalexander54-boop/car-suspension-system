# car-suspension-system
See the swing of the car !

Here, we can see that the mercedes benz car and rolls royce car, how much they go up and down when going over a normal bump!!!
these diagrams are obtained using this differential equation
$(mx'')+(cx')+(kx)=0$
that
$acceleration=x''=d^2x/dt^2$
and
$speed=x'=dx/dt$
